# OPERATION: PARIS

A cinematic weekend itinerary for Paris, styled as a classified intelligence dossier. Two days, two sides of the city — the iconic and the underground. Covers Sciences Po, the Catacombes, a natural wine cave, Paradis Latin, and more, with a Leaflet map, Google Calendar export, redacted hover text, and a "burn this document" easter egg. Built with plain HTML, CSS, and JavaScript — no framework, no build step.
